## 🌟 Overview

### The Problem
Managing orders on WhatsApp is a chaotic manual process. Sellers spend hours copying addresses, tracking prices, and manually messaging customers for status updates. One missed message means a lost sale or a frustrated customer.

### The Solution
**OrderMesh** is a professional-grade automation engine that bridges the gap between WhatsApp conversations and your order management portal. It automatically identifies order details from messages, saves them to a secure database, and triggers automated follow-up notifications as the order moves through your pipeline.

---

## 🚀 Key Features

- **⚡ Intelligent Order Parsing:** Instantly extracts Customer Name, Address, Product, Price, and Contact info from WhatsApp messages starting with `#order`.
- **🔄 Live Portal Synchronization:** Every order is pushed in real-time to a private Firebase-powered dashboard for fulfillment.
- **🔔 Automated Customer Journey:** Sends instant WhatsApp notifications for critical milestones:
  - ✅ Order Confirmation
  - 🚢 Shipped Updates
  - 🚚 Out for Delivery (with Courier Details)
  - 🎉 Delivery Success
  - ❌ Cancellation Alerts
- **🛠️ Smart Connectivity:** Advanced session management to keep the bot online 24/7, even on low-resource servers.
- **📱 Localization Ready:** Optimized for international phone formats (special focus on Pakistan/South Asia regions).
- **🛡️ Reliable Performance:** Built on WPPConnect for high-speed, stable WhatsApp automation.

---

## 🛠️ Tech Stack

- **Engine:** Node.js (V8 Runtime)
- **WhatsApp Integration:** WPPConnect (Enterprise-level automation)
- **Database & Real-time Sync:** Firebase Firestore
- **Automation Core:** Puppeteer (Headless Browser Management)
- **Environment:** Designed for Ubuntu/Linux Server Deployment
